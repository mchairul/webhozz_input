# webhozz_input

Simple Widget Input.

## Installation

You just need to add `webhozz_input` as a [dependency in your pubspec.yaml file](https://flutter.io/using-packages/).

```yaml
dependencies:
  webhozz_input: ^0.0.1
```

## Example

In these examples, show input with InputDecoration circular.

```dart
WebhozzInput(
  type: '',
  label: Text('label'),
  obscureText: false
);
```
